# git使用教程


### 三板斧打完就跑

```git
# 获取项目
git clone https://github.com/schacon/simplegit-progit

# 查看状态
git status

# 选择需要的文件
git add 文件名(“.”目录下所有文件)  -- 添加
git rm  文件名  -- 删除

# commit -m ＋注释
git commit -m "update"

# push
git push 
```



### 同步

```
# 查看有木有更新 
git fetch

# 有，就搞下来
git pull 
```
