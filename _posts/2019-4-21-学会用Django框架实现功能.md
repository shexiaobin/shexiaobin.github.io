### 2.7 在Model中创建数据表

#### 2.7.1 models.py

```python
class People(models.Model):
# null表示数据库可以没有数据；
# blank名字什么都不填也没有问题；
# max_length长度不超过200
	name = models.CharField(null=True, blank=True, 
                        max_length=200)
	job = models.CharField(null=True, blank=True, 
                       max_length=200)
#admin后台中数据名称显示为name
	def __str__(self):
        return self.name
```

#### 2.7.2 合并数据库

```python
python3 manage.py makemigrations
python3 manage.py migrate
#每次model层有改动都要输入这两行合并数据库，切记切记
```
